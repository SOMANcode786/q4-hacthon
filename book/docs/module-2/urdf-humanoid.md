# URDF for Humanoid Robots

This module covers the Unified Robot Description Format (URDF) and how to model humanoid robots for simulation and control in ROS 2.

## Learning Objectives

- Understand URDF structure and syntax
- Model humanoid robot kinematics
- Define joints, links, and physical properties
- Create collision and visual geometries
