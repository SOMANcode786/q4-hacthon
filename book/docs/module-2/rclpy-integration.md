# Python Integration with rclpy

This module covers how to use ROS 2 from Python using the rclpy client library, which provides a Python API for ROS 2.

## Learning Objectives

- Understand the rclpy client library
- Learn how to create ROS 2 nodes in Python
- Implement publishers, subscribers, and services
- Work with ROS 2 parameters and actions in Python
