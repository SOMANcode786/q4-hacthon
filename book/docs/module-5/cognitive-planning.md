# Cognitive Planning for Robots

This module covers cognitive planning systems that enable robots to reason about complex tasks, break them down into executable steps, and adapt to changing conditions.

## Learning Objectives

- Understand cognitive architectures for robotics
- Learn about task and motion planning
- Explore reasoning under uncertainty
- Implement adaptive planning systems
