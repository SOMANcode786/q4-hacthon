# AI-Humanoid-Robotics Project

Welcome to the AI-Humanoid-Robotics project! This repository contains educational materials about physical AI and humanoid robotics.

## About

This project demonstrates the integration of AI with physical systems through:
- ROS 2 robotic frameworks
- NVIDIA Isaac for AI-powered robotics
- Gazebo simulation environments
- Vision-Language-Action systems
