# Navigation 2 for Humanoids

This module covers ROS 2 Navigation (Nav2) system adapted for humanoid robots, including path planning, obstacle avoidance, and locomotion control.

## Learning Objectives

- Understand Nav2 architecture and components
- Adapt navigation for humanoid locomotion
- Configure planners for bipedal movement
- Implement obstacle avoidance for humanoids
